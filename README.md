## Getting Started

用Node.js + Express框架練習的作品，內容主要為餐廳清單及其詳細資訊。

(/public/webpic.png)

## Features
* 依照餐廳名稱進行搜尋
* 檢視餐廳詳細介紹包含類別、電話、圖片及地址連結

## Environment Setup
* Node.js

## Built With & Tools
* Express framework
* Express-handlebars
* Static Files include Bootstrap, jQuery, Popper.js and fontawesome
